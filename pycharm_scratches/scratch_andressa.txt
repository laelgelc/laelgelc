Tendo como objetivo produzir o estudo proposto para o IVACS 2026, acordamos os seguintes passos iniciais:

1. Coleta de dados brutos das seguintes comunidades através do Reddit Data API. Utilizaremos IA para planejar uma extração de dados que seja ética e viável.
https://www.reddit.com/r/lonelywomen
https://www.reddit.com/r/solitude
https://www.reddit.com/r/loner
https://www.reddit.com/r/loneliness/
https://www.reddit.com/r/lonely/
2. Quantidade de postagens do subcorpus humano: Deverá ser validada com o Professor Tony. Para fins de um processamento piloto, utilizaremos a regra prática de 5 textos (postagens) por variável lexical. Com 1000 variáveis lexicais, necessitaremos de 5000 postagens. Os corpora gerados por IA (o guiado por resumo e o genérico) refletirão as postagens humanas, formando um corpus com três subcorpora balanceados e 15000 postagens. Podemos manter a mesma quantidade de 1000 variáveis lexicais para o corpus consolidado.
3. Engenharia de prompts: Os prompts deverão ser projetados para extrair do modelo de raciocínio os discursos acerca de solidão no cenário de produção genérica e naquele guiado por resumo. Será necessário projetar 3 tipos de prompt:
- Produção genérica
- Prompt para geração de postagem
- Produção guiada por resumo
- Prompt para geração de resumo
- Prompt para geração de postagem
